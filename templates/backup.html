Les backups sont stockés localement au format JSON et peuvent être téléchargés ou supprimés à tout moment.
Instructions

Pour sauvegarder un serveur Discord, vous devez :

  1. Avoir un token Discord valide (celui de votre compte personnel)
  2. Avoir les permissions nécessaires sur le serveur à sauvegarder
  3. Connaître l'ID numérique du serveur
Note : Votre token n'est jamais stocké sur notre serveur. Il est uniquement utilisé pour effectuer les opérations de sauvegarde.
